## Crawling Twiiter with apify
Crawling Twiiter with apify and  Message with Kafka 

<br>


### Installation

Yang perlu di siapkan 

```bash
    pip install -r requirements.txt
```


<br>


#### How to use

1. copy the .env

```bash
cp .env.example .env
```

2. run the script

```bash
python run.py
```
